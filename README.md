# AutoApply Bots

Job application automation scripts for Wellfound and Naukri.

---

### Wellfound — [`Wellfound/`](./Wellfound/)

Browser console script. No install needed. Runs on [wellfound.com/jobs](https://wellfound.com/jobs).  
Repo: [github.com/Vedmeena21/Wellfound_AutoApply](https://github.com/Vedmeena21/Wellfound_AutoApply)

---

### Naukri — [`Naukri/`](./Naukri/)

Python + Selenium bot. Runs on [naukri.com](https://www.naukri.com).  
Repo: [github.com/Vedmeena21/Naukri-autoapply-bot](https://github.com/Vedmeena21/Naukri-autoapply-bot)

---

Each subfolder has its own README with full setup and usage instructions.
